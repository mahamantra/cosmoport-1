# cosmoport

JavaRush online internship test task #2 

## Getting Started

This is my implementation of JavaRush summer 2019 online internship test task #2.

### Technologies used:

* Spring MVC
* Spring Data JPA

### Required software

* MySQL server
* TomCat9 server

### Installing

```
mvn clean install -U
```

## Running the tests

```
mvn test
```

## Built With

* [Maven](https://maven.apache.org/) - Dependency Management
